Sobre

Esse é um projeto de Single Page Application (SPA) que simula um site de jogos Flash.

Jogos Flash eram muito populares nos anos 2000 e 2010. Eram jogos leves, que rodavam no navegador e eram fáceis de compartilhar. Esse projeto é uma forma de trazer um pouco dessa nostalgia de volta.

Acessar Github

Como foi feito?

O projeto foi feito somente com HTML, CSS e JavaScript simples, sem o uso de frameworks ou bibliotecas como o React ou Vue.

Você pode navegar entre as páginas e jogos sem recarregar a página. Isso é feito utilizando JavaScript para interceptar os cliques nos links e fazer requisições (hashchange) para carregar o conteúdo dinamicamente.

E sobre os jogos Flash?

O projeto utiliza da biblioteca Ruffle para emular os jogos Flash. A Ruffle é uma biblioteca que interpreta e executa os jogos Flash em tempo de execução.

Como faço para jogar outros jogos Flash?

Basta clonar esse repositório e adicionar os jogos que deseja jogar na pasta /assets/games. Os jogos devem ser arquivos SWF.

Após adicionar os jogos, você deve atualizar o arquivo data/games.js com as informações dos jogos adicionados.

Para adicionar um novo jogo, você deve adicionar um novo objeto no array games com as seguintes informações:

games: [{id: 1, name: "Nome do jogo", img: "nome-da-imagem.webp", fileName: "nome-do-jogo.swf",},]
