Bem vindo!

Em 2020, a Adobe anunciou que o Flash Player seria descontinuado. Com isso, muitos jogos e animações em Flash deixaram de ter suporte nos navegadores.

Criaram-se então alternativas para emular os jogos Flash. Este projeto utiliza uma dessas soluções, a Ruffle, para emular os jogos Flash e permitir que você jogue novamente.

Jogos disponíveis

O projeto conta com 10 jogos disponíveis para jogar.

Não é necessário instalar nada, porém por se tratarem de jogos voltados ao acesso via computador, é recomendado que você acesse o site pelo seu PC para uma melhor experiência.

Sobre o projeto

O projeto não utiliza de um servidor para armazenar os jogos por se tratar apenas de uma Prova de Conceito, por isso existem poucos jogos disponíveis.

Mas fique a vontade para contribuir com o projeto, basta acessar o repositório no GitHub, clonar o projeto e adicionar mais jogos na lista!
